# Synvex Project Management Dashboard

A modern, highly-responsive, dark-themed SaaS dashboard designed for agile project and team management. Built with a focus on performance, aesthetics, and user experience, this platform provides real-time visibility into project statuses, task delegation, and team collaboration.

## 🚀 Features

- **Centralized Dashboard**: A bird's-eye view of your workspace with real-time statistics, progress indicators, and a recent activity feed.
- **Project Tracking**: Manage multiple projects simultaneously. Track completion percentages, deadlines, and milestones.
- **Task Management**: Deep-dive into individual tasks with Kanban-style organization (To Do, In Progress, Done). Features a rich discussion panel for task-specific communication.
- **Team Collaboration**: Easily invite members, assign tasks, and monitor team workload.
- **Premium Dark UI**: Built with a custom, high-contrast dark theme utilizing soft semantic colors and sleek visual hierarchy.

## 💻 Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Routing**: React Router v6
- **State Management**: Redux Toolkit (RTK)
- **Styling**: Tailwind CSS v4 (Custom semantic CSS variables)
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

## 🛠️ Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/iam-Jam24/Sem2-OJT.git
   cd Sem2-OJT
   ```

2. **Install Dependencies**:
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Start the Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy
This project ditches the standard neo-brutalist or flat light modes in favor of a permanent, immersive dark mode. By centralizing our design tokens in pure CSS variables, components instantly adapt their border, surface, and text contrasts while maintaining absolute consistency.
